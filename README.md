# 캡스톤 프로젝트 소개 웹 (EC2 + Docker, 또는 로컬)
본 저장소는 캡스톤 프로젝트 단일 소개 페이지와 API를 제공합니다.

## 1. 작품 주제 (Subject)
- 제목 : Mozik (모자이크 시스템)
- 요약 : 영상속에서 노출되는 민감한 정보를 보호하기 위한 어플리케이션

## 2. 실용적 근거 (Rationale)
- 문제 : 영상 속에서 노출되는 민감한 정보를 가려야 하기에 수동으로 모자이크 처리하는 데 많은 시간과 노력이 소요됨
- 근거 (수치/ 설문/ 사례) : 영상속에서 모자이크처리를 하지 않아 초상권을 침해당하는 일이 발생, 개인 정보 침해 신고 및 상단 건수가 꾸준히 높음, 수동 모자이크 처리에 많은 시간과 노력이 소요됨
- 기대 가치 : 민감한 정보 보호를 통해 개인의 프라이버시를 지키고, 초상권 침해 문제를 예방할 수 있으며, 영상 편집의 효율성을 높여 시간을 절감할 수 있음

## 4. 구현환경 (Environment)
- Front-End : Mobile : andriod studio, Kotlin | Web service :HTML, CSS, FastAPI, Flask
- Back-End : Back-End(백엔드):Python, Flask,FastAPI
- Object Analysis(객체분석):Tensorflow, OpenCV, Yolov5, MediaPipe, FFmpeg 
- Database(데이터베이스): KT Cloud ,MySQL

- 최정환 — Backend(프로젝트 설계 및 연결, 인공지능 모델 학습)
